# AI-Driven Banking Customer Analytics Dashboard

## 📌 Project Overview
This project analyzes ICICI Bank customer data using Excel, Power BI, and Microsoft Copilot to generate insights on financial health, risk levels, loan exposure, and customer behavior.

## 🛠 Tools Used
- Microsoft Excel
- Power BI
- Microsoft Copilot Studio
- AI Chatbot

## 🤖 AI Integration
A custom AI agent was developed using Copilot Studio with dataset integration to provide real-time insights and recommendations.

## 📊 Key Features
- Customer Segmentation Analysis
- Risk Categorization (Low, Medium, High)
- Loan Exposure Monitoring
- Active vs Inactive Accounts
- AI-generated Insights
